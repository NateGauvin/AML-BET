AML-BET (Acute Myeloid Leukemia Biomarker Evaluation Tool)

This repository is for my Eastern Connecticut State University Honors Scholars Program thesis project.
Included is my thesis proposal, as well as the prototype/proof-of-concept for AML-BET.
All future work on this thesis will be recorded here. Further progress will begin once the Spring 2025 semester begins, and will conclude at the end of Fall 2025.
This project uses R, R Shiny, R Golem, MongoDB, and Docker.

I recommend reading the proposal for an in-depth overview of AML-BET and its significance.
